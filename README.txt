ATS Resume Checker
==================

Files:
- index.html  -> main app

How to use:
1. Open index.html in any browser.
2. Paste resume text.
3. Paste job description.
4. Click "Analyze Resume".

What it checks:
- ATS score out of 100
- Keyword match with job description
- Section coverage
- Format health
- Readability
- Improvement suggestions

Notes:
- This is a frontend-only product, so it is easy to sell/demo.
- Best for Gumroad, Fiverr, and direct client demos.
- You can later upgrade it with PDF parsing, AI suggestions, login, or backend storage.
